# xiuxiuzhushou

`xiuxiuzhushou` is being rebuilt as a local-first study assistant with a harness-style workflow.

## Commands

```bash
npm install
npm run dev
npm run check
```

## Repo Flow

- `AGENTS.md` is the entry point for agents.
- `docs/` holds product, architecture, and verification rules.
- `harness/` holds persistent execution memory.
- `artifacts/evidence/latest/` is generated by `npm run verify`.

## Current V1

- View plans, habits, rewards, and star balance.
- Add a new study plan.
- Complete a plan or habit and update stars.
- Redeem a reward with balance checks.
